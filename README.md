# mdarafat — Personal Portfolio Website

A premium personal portfolio website for MD Arafat (Android & WordPress Developer).

## ✨ Features
- Dark premium design with cyan/purple/coral color palette
- Custom animated cursor
- Smooth scroll reveal animations
- Animated stat counters
- Skill progress bars
- Project cards with hover effects
- Services section with animated arrows
- Contact form
- Fully responsive (mobile-friendly)
- Vercel-ready static deployment
